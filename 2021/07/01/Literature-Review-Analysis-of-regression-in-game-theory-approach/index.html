Literature Review: Analysis of regression in game theory approach

Author: John Doe Date: July 1, 2021  16:27:46

paper origin: https://onlinelibrary.wiley.com/doi/abs/10.1002/asmb.446

Problem Statement: Why traditional analysis algorithms don’t work well?

Multicollinearity

Multicollinearity is a linear association among more than two explanatory variables in a multiple regression model. More than two variables are perfectly collinear if there is an exact linear relationship among them.

Multicollinearity causes a reduction in statistical power, or the ability of statistical tests to detect true differences in the population.

Net Effects

Net effects is a technique for regression analysis to evaluate the relative importance of the predictors. The net effect of a predictor is a combination of the direct (regression coefficients) and the indirect (correlation among variables) effects.

problems:

  • The net effects are influenced by collinear redundancy

Find Optimal Subset

A good technique is finding the most necessary variables for the equation.

problems:

  • Since the power reduction caused by multicollinearity, it is hard to find the optimal model. Also, it is hard to determine which is the best superior model.

A good solution for that is using some models like Bagging, bootstrap aggregation, boosting and bundling. But inference about specific variables is difficult for those models.

What is Shapley Value imputation?

Shapley Value imputation is an analysis and decision tool in the field of co-operative games.

Shapley Value can be calculated by the results of the regression calculation, which reshapes the regression net effects and coefficients.

Why to use Shapley imputation?

  1. It is an axiomatic approach instead of an heuristic procedure

Algorithm details

for regression like:
$$ y_i=\beta_1x_{1i}+\beta_2x_{2i}+\beta_3x_{3i} + … + \beta_nx_{ni} + e_i$$
where y and xs are standardized variables. e is a Guassian Random Er ror and i is a number of observation.

$$S^2 = e^Te = (y-Xb)^T(y-Xb)=1-2b^Tr + b^TCb$$

where $C$ is the correlations matrix. and $r$ is the sample correlations between each $x_j$ and the dependent variable $y$.

$$C=X^TX$$

$$Cb = r$$

Minimizing $S^2$ is equaivalent to maximize the multiple determination $R^2$

$$R^2 = 1-S^2=2b^Tr - b^TCb=\sum\limits_jb_j(2r-Cb)_j$$

Since $Cb=r$

$$R^2=b^Tr=\sum\limits_jb_jr_j$$

And net effects (NEF) of each 4th regressor is:

$$NEF_j=b_jr_j$$

There are other more reasonable indices for evaluation of predictor importance.

For example, the share of $x_j$ could be defined by the square of the partial correlation between $y$ and $x_j$ with fixed other x’s

$$R^2_{yj}=(R^2 - R^2_{-j})/(1-R^2_{-j})$$

where $R^2$ demotes to multiple determination as mentioned above and $R^2_{-j}$ denotes to multiple determination without $x_j$

Also, we can use

$$U_j=(R^2-R^2_{-j})$$

when $R_{-j}<<1$, $U_j$ is converged to $R^2_{yj}$

Here the original definition for Shapley Value imputation is

$$S_j=\sum\limits_{all\ M} \gamma_n(M)[v(M\cup{j})-v(M)]$$

with weights of proportions $\gamma_n(M)$

$$\gamma_n(M)=\frac{m!(n-m-1)!}{n!}$$

and $v()$ which is the characteristic function used for estimation of utility for each coalition.

$M$ denotes to a coalition without the $j$th participant. and $M\cup{j}$ denotes to a coaltion including the $j$th participant.

Here we define the characteristic function $v()$ via the $R^2$ values estimated by the results of regression modeling.
For example, if there are 5 elements, we have:

$$v(0)=0, v(A)=R^2_A, v(AB)=R^2_{AB}, …, v(ABCDE)=R^2_{ABCDE}$$

Also, we can simiplify the formula in this way:

$$S_j=\sum\limits_{all\ M\ including\ j}\gamma_n(M)U_M$$

And

$$S_A = \frac{R^2_A-\bar{R}1^2}{n-1}+\frac{\bar{R}^2{A^{}}-\bar{R}2^2}{n-2}$+\frac{\bar{R}^2 {A^{}}-\bar{R}3^2}{n-3}$+…+\frac{\bar{R}^2{A^{}}-\bar{R}1^2}{n-(n-1)}+\frac{R^2{AB…N}}{n}$$
